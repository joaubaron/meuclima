# Meu Clima ☀️

Aplicativo PWA de previsão do tempo com informações detalhadas, gráficos e sugestões personalizadas.

## Funcionalidades

- 🌡️ Temperatura atual com ícone dinâmico
- 💧 Umidade, precipitação e vento
- 📈 Gráficos das próximas 24h
- 🌙 Fase da lua com iluminação
- 🍂 Estação do ano e contagem
- 👕 Sugestão de vestuário
- 🍽️ Sugestão de receita
- 🔔 Alertas climáticos
- 📍 Geolocalização
- 💾 Cache offline
- 🔄 Pull to refresh
- 📱 Instalável (PWA)

## Tecnologias

HTML5 / CSS3 / JavaScript (ES6+) / Chart.js / WeatherAPI / Nominatim

## Estrutura


## Uso

1. Acesse e permita a localização
2. Veja o clima atual e previsões
